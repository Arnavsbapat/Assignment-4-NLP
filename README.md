# Assignment-4-NLP
NLP preprocessing and text classification using NLTK, spaCy, TF-IDF, and Naive Bayes on movie reviews dataset.
# NLP Preprocessing and Text Classification

## 📌 Overview
This project demonstrates Natural Language Processing (NLP) techniques and text classification using machine learning. The goal is to preprocess textual data and build a model that can classify movie reviews as positive or negative.

## 🎯 Objective
- Apply NLP preprocessing techniques
- Convert text into numerical form using vectorization
- Build and evaluate a classification model

## 🧠 Techniques Used
- Tokenization
- Stopword Removal
- Lemmatization (spaCy)
- Stemming (Porter Stemmer)

## 🔢 Text Vectorization
- Count Vectorizer (Bag of Words)
- TF-IDF (Term Frequency - Inverse Document Frequency)

## 🤖 Machine Learning Model
- Multinomial Naive Bayes

## 📊 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

## 📁 Dataset
- NLTK Movie Reviews Dataset (2000 labeled reviews)

## ⚙️ Tools & Libraries
- Python
- NLTK
- spaCy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## 🚀 Workflow
1. Install and import libraries
2. Load dataset
3. Perform preprocessing
4. Apply vectorization
5. Train model
6. Evaluate performance
7. Test with custom inputs

## 📈 Results
The model achieves good accuracy using TF-IDF and Naive Bayes, demonstrating effective text classification performance.

## 🧪 Sample Output
- Positive Review → POSITIVE
- Negative Review → NEGATIVE

## 📌 Conclusion
This project shows how NLP preprocessing and machine learning can be combined to build an effective sentiment analysis system.
